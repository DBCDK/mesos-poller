Mesos-Poller

Probes Mesos for information about instances and their endpoints for a given web-service.

Path: "http://this.host/status?app=Full-Mesos-App-Path"
fetches information about the given Application, and probes each in turn, using the mesos configured status-endpoint.

Path: "http://this.host/status?app=Full-Mesos-App-Path&part=Status-Endpoint"
fetches information about the given Application, and probes each in turn, using the given status-endpoint.

app
The full path of the application in Mesos (e.g. prod/search/cisterne/opensearch/opensearch-5.0)
part
The status part of the applications URL (e.g. /opensearch/?HowRU)

Returns the result in form of JSON
Example:
{"num-endpoints": 5, "app": "prod/search/cisterne/opensearch/opensearch-5.0", "num-ok": 5, "status": "OK"}
